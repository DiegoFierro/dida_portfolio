Python Fundamentals Documentation

Python is a high-level, interpreted, general-purpose programming language known for its readability and simplicity. Its design philosophy emphasizes code readability with the use of significant indentation. It supports multiple programming paradigms, including object-oriented, imperative, and functional programming.

Core Syntax and Data Types

Python is dynamically typed, meaning you don't need to declare the variable type explicitly. Variables are created when you first assign a value to them.

Comments in Python are denoted by the hash symbol (`#`). Anything after the `#` on a line is ignored by the interpreter.

# This is a single-line comment
age = 30 # Integer variable
name = "Alice" # String variable
pi = 3.14 # Float variable
is_active = True # Boolean variable

Python comes with several built-in basic data types:

  • Integers (`int`): Whole numbers (e.g., `10`, `-5`, `100000`).
  • Floats (`float`): Decimal numbers (e.g., `3.14`, `-0.5`, `2.718`).
  • Strings (`str`): Sequences of characters enclosed in single or double quotes (e.g., `"hello world"`, `'Python'`).
  • Booleans (`bool`): Represent truth values, either `True` or `False`.

Common operators include arithmetic, comparison, and logical operators:

# Arithmetic Operators
result_sum = 5 + 3   # 8
result_prod = 4 * 2  # 8
result_div = 10 / 3  # 3.333...
result_floor_div = 10 // 3 # 3
result_mod = 10 % 3  # 1
result_exp = 2 ** 3  # 8

# Comparison Operators
is_equal = (5 == 5) # True
is_not_equal = (5 != 3) # True
is_greater = (7 > 4) # True

# Logical Operators
logical_and = (True and False) # False
logical_or = (True or False) # True
logical_not = (not True) # False
Control Flow

Control flow statements dictate the order in which instructions are executed. Python uses indentation to define code blocks.

Conditional Statements (`if`, `elif`, `else`)

These statements allow you to execute different blocks of code based on whether certain conditions are met.

age = 20
if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teenager")
else:
    print("Child")

Loops (`for`, `while`)

Loops are used to repeatedly execute a block of code.

The `for` loop iterates over a sequence (like a list, tuple, string, or range).

# For loop with range
for i in range(3): # Iterates from 0 to 2
    print(f"Iteration {i}")

# For loop over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}")

The `while` loop executes a block of code as long as a specified condition is true.

count = 0
while count < 3:
    print(f"Count is {count}")
    count += 1

`break` and `continue` statements can alter loop behavior:

  • `break`: Terminates the loop entirely.
  • `continue`: Skips the rest of the current iteration and proceeds to the next.
Data Structures

Python provides several built-in data structures to organize and store data efficiently.

Lists (`list`)

Ordered, mutable (changeable) collections of items. Enclosed in square brackets `[]`.

my_list = [1, "hello", 3.14, True]
print(my_list[0]) # Access by index: 1
my_list.append("new item") # Add item
my_list[1] = "world" # Modify item

Tuples (`tuple`)

Ordered, immutable (unchangeable) collections of items. Enclosed in parentheses `()`.

my_tuple = (1, 2, "three")
print(my_tuple[2]) # Access by index: "three"
# my_tuple[0] = 5 # This would raise an error (immutable)

Dictionaries (`dict`)

Unordered, mutable collections of key-value pairs. Enclosed in curly braces `{}`.

my_dict = {"name": "Alice", "age": 30, "city": "New York"}
print(my_dict["name"]) # Access by key: "Alice"
my_dict["age"] = 31 # Modify value
my_dict["occupation"] = "Engineer" # Add new key-value pair

Sets (`set`)

Unordered, mutable collections of unique items. Duplicates are automatically removed. Enclosed in curly braces `{}` (or using `set()`).

my_set = {1, 2, 3, 2, 1} # Result: {1, 2, 3}
print(my_set)
my_set.add(4) # Add item
my_set.remove(1) # Remove item
Functions

Functions are reusable blocks of code that perform a specific task. They help in organizing code and promoting reusability.

Defining Functions (`def`)

Functions are defined using the `def` keyword, followed by the function name, parentheses for parameters, and a colon.

def greet(name):
    """This function greets the person passed in as a parameter."""
    return f"Hello, {name}!"

message = greet("Bob")
print(message) # Output: Hello, Bob!

Functions can accept parameters and return values using the `return` keyword.

Lambda Functions

Lambda functions are small, anonymous functions defined with the `lambda` keyword. They can take any number of arguments but can only have one expression.

add = lambda x, y: x + y
print(add(5, 3)) # Output: 8

multiply = lambda a, b: a * b
print(multiply(4, 6)) # Output: 24
Modules and Packages

Python's modular design allows you to organize code into reusable files and directories.

Modules

A module is simply a Python file (`.py`) containing Python code. You can import modules to use their functions, classes, and variables.

import math
print(math.sqrt(16)) # Output: 4.0

from datetime import datetime
print(datetime.now()) # Prints current date and time

Packages

A package is a collection of modules organized in directories. It helps in structuring larger applications.

The `import` statement is used to bring modules or specific components from modules into the current scope.

  • `import module_name`: Imports the entire module.
  • `from module_name import specific_item`: Imports only a specific function, class, or variable.
  • `import module_name as alias`: Imports a module with a shorter alias.
Object-Oriented Programming (OOP) Concepts

Python fully supports Object-Oriented Programming, a paradigm based on the concept of "objects", which can contain data and code.

Classes and Objects

A class is a blueprint for creating objects. An object is an instance of a class.

class Dog:
    # The __init__ method is the constructor
    def __init__(self, name, breed):
        self.name = name    # Attribute
        self.breed = breed  # Attribute

    # A method
    def bark(self):
        return f"{self.name} says Woof!"

# Creating an object (instance) of the Dog class
my_dog = Dog("Buddy", "Golden Retriever")
print(my_dog.bark()) # Calling a method: Buddy says Woof!
print(my_dog.name) # Accessing an attribute: Buddy

Key OOP principles include:

  • Encapsulation: Bundling data (attributes) and methods that operate on the data within a single unit (class).
  • Inheritance: Allows a class (child/subclass) to inherit attributes and methods from another class (parent/superclass), promoting code reuse.
  • Polymorphism: The ability of different objects to respond to the same method call in their own way, often through method overriding.
File Handling

Python provides built-in functions to interact with files on your system, allowing you to read from and write to them.

The `open()` function is used to open a file, specifying the file path and the mode (e.g., `'w'` for write, `'r'` for read, `'a'` for append).

# Writing to a file (creates if not exists, overwrites if exists)
with open("example.txt", "w") as file:
    file.write("Hello, Python file handling!\n")
    file.write("This is a new line.")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
    print("File content:")
    print(content)

# Appending to a file
with open("example.txt", "a") as file:
    file.write("\nAppending another line.")

The `with` statement is highly recommended for file operations as it ensures the file is automatically closed, even if errors occur.

Exception Handling

Exception handling allows you to gracefully manage errors and unexpected events that occur during program execution, preventing your program from crashing.

The `try`, `except`, and `finally` blocks are used for this purpose:

  • `try`: The code that might raise an exception is placed here.
  • `except`: If an exception occurs in the `try` block, the code in the `except` block is executed. You can specify the type of exception to catch.
  • `finally`: The code in the `finally` block is always executed, regardless of whether an exception occurred or was handled.
try:
    num1 = int(input("Enter a number: "))
    num2 = int(input("Enter another number: "))
    result = num1 / num2
    print(f"Result: {result}")
except ValueError:
    print("Invalid input. Please enter integers only.")
except ZeroDivisionError:
    print("Error: Cannot divide by zero!")
except Exception as e: # Catch any other unexpected errors
    print(f"An unexpected error occurred: {e}")
finally:
    print("Execution of the try-except block is complete.")
Popular Libraries and Ecosystem

One of Python's greatest strengths is its extensive ecosystem of third-party libraries and frameworks, which significantly extend its capabilities across various domains.

  • NumPy: Essential for numerical computing, especially for working with large, multi-dimensional arrays and matrices.
  • Pandas: A powerful library for data manipulation and analysis, providing data structures like DataFrames.
  • Matplotlib/Seaborn: Widely used for creating static, animated, and interactive visualizations in Python.
  • Scikit-learn: A comprehensive library for machine learning, offering various classification, regression, and clustering algorithms.
  • Django/Flask: Popular web development frameworks for building robust and scalable web applications.
  • Requests: A simple yet elegant HTTP library for making web requests.
  • Beautiful Soup: Used for parsing HTML and XML documents, often for web scraping.

These are just a few examples; the Python Package Index (PyPI) hosts thousands of libraries for almost any task imaginable.

Why Python?

Python's popularity stems from several key advantages that make it a preferred choice for developers and organizations alike.

  • Readability and Simplicity: Its clear, concise syntax makes it easy to learn, read, and maintain, leading to faster development cycles.
  • Versatility: Python is a general-purpose language applicable in a vast array of fields, including web development, data science, machine learning, artificial intelligence, automation, scripting, scientific computing, and more.
  • Large Community and Ecosystem: A vibrant and active community contributes to a wealth of resources, tutorials, and an enormous collection of third-party libraries and frameworks.
  • Cross-Platform Compatibility: Python code can run seamlessly on various operating systems, including Windows, macOS, and Linux, without modification.
  • High-Level Language: It abstracts away complex low-level details of memory management and hardware interaction, allowing developers to focus more on problem-solving.

These factors combine to make Python an incredibly powerful and accessible language for beginners and experienced developers alike.
